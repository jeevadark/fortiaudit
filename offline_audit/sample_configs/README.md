# Sample FortiGate Configurations

Place sanitized sample configurations here for testing.

## How to Add Samples

1. Export config from FortiGate
2. Sanitize (remove passwords, real IPs, company names)
3. Save as `.conf` file
4. Test with offline auditor

## Naming Convention

- `fortigate_small_office.conf` - Small setup (< 10 policies)
- `fortigate_enterprise.conf` - Large setup (100+ policies)  
- `fortigate_ha_cluster.conf` - HA configuration example
